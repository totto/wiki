# wiki.totto.org

Personal wiki and blog of Thor Henning Hetland, built with [MkDocs Material](https://squidfunnel.github.io/mkdocs-material/).

## Content

- **About** -- Bio, CV (English & Norwegian), project history, presentations, organizations
- **Blog** -- Cloud computing posts from 2009
- **Interests** -- Sci-fi reading lists (2012-2015), food & wine, tools
- **Notes** -- Architecture vs Agile panel, Twitter archive

## Development

```bash
pip install mkdocs-material
mkdocs serve
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.

Custom domain: `wiki.totto.org`

## Origin

Content migrated from the `~totto` personal wiki space in [Cantara/wiki](https://github.com/Cantara/wiki).
