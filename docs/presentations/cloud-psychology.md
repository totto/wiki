# Cloud Psychology: Why Many Businesses Will Go Out of Business

**Date:** ~2010
**Slides:** [slideshare.net/totto](https://www.slideshare.net/totto/cloud-psychology)

The psychological barriers preventing organisations from adopting cloud computing. Fear, Uncertainty, and Doubt (FUD) create resistance, preventing businesses from realising strategic value. Real project examples illustrate how to remove cloud psychology from key decisions and capture the actual economics of scale that cloud infrastructure offers.
